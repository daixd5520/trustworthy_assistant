# 字符串分割回溯套路

## Word Break II

### 核心框架

回溯三步：**选、收、撤**。

```
dfs(start):
    if start == len(s):         # 递归出口
        return [""]             # 返回空串，表示这条路通了
    
    for end in range(start+1, len(s)+1):
        sub = s[start:end]
        if sub not in vocab: continue
        
        for suffix in dfs(end):  # "选" + 递归
            if suffix == "":
                result.append(sub)
            else:
                result.append(sub + " " + suffix)
    
    return result
```

### 关键点

1. **递归出口**：走到末尾返回 `[""]`，不是 `[]`
2. **返回值的意义**：`dfs(end)` 返回从 end 开始的所有完整方案
3. **每层做的事**：把当前词拼到下一层每种方案前面
4. **记忆化**：用 memo 存 `start` 位置的结果，避免重复计算

### 一句话

> 出口在最开头判断，处理逻辑在递归调用之后
