# 食物热量估算

识别食物图片，估算卡路里，自动记入账本。

## 工作方式

1. 你发一张食物照片（饭菜、零食、外卖、餐盘都可以）
2. 我用 vision 识别主要食物和份量
3. 估算每个组成部分的热量，给出整餐区间
4. 可选：自动记入账本（记到 `food` 分类）

## 估算说明

这是**保守估算**，不是精确值：
- 基于常见餐厅/家庭份量
- 基于典型烹饪方式（不特别精确）
- 误差可能 ±20-30%，重油菜/酱料多时更大
- 奶茶、炸物、火锅等浮动大的会特别标注

## 估算参考（常见经验值）

| 食物 | 热量区间 |
|------|----------|
| 一小碗米饭 | 180-230 kcal |
| 一拳头熟面 | 220-320 kcal |
| 一份炒菜（200g） | 100-200 kcal |
| 一块煎鸡腿 | 200-350 kcal |
| 一份炒饭/盖饭 | 500-900 kcal |
| 一杯奶茶（中杯） | 250-500 kcal |
| 一份炸鸡（整只） | 700-1200 kcal |

## 输出格式

识别结果 → 热量估算 → 主要来源 → 不确定性说明

## 使用示例

- "记一下这顿热量" + 发图
- "帮我看看这碗面多少卡"
- "这个外卖出估算记到账本里"
