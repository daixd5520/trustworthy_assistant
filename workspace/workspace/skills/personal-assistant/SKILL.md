# 个人生活助手

随时待命，帮你打理日常琐事。

## 可用功能

### 1. 定时提醒
**显式请求**："30分钟后提醒我开会"、"下午3点提醒我吃药"

**隐式意图（自动识别）**：当你说这些关键词时，我会自动提取时间并设置提醒：
- "先这样" / "先玩了" / "先休息一下" / "先走一步"
- "玩 X 分钟手机" / "休息 X 分钟" / "摸 X 分钟鱼"

隐式场景示例：
- "想玩五分钟手机" → 5分钟后提醒"该干活了"
- "先这样，10分钟后叫我" → 10分钟后提醒

### 2. 记账
- "记一笔支出 32 元，餐饮" — 快速记消费
- "今天花了多少" — 查看当日支出
- "这周账单" / "本月统计" — 按周期查账

### 3. 记忆
- "帮我记住我最喜欢孙燕姿的《银泰》" — 存入长期记忆
- "我之前说过什么" / "你记得我吗" — 召回记忆

### 4. 每日聊天摘要
每天晚上 10:30 之后，我会自动整理当天我们聊了啥发给你，包括：
- 讨论的话题
- 做过的决定
- 待办事项提醒

### 5. 主动推送配置
需要的话可以说：
- "每天晚上 11 点发我当日账本"
- "每周日给我发周报"
- "每月 1 号发我月账单"
